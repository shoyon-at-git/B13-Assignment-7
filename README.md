# 🌿 KeenKeeper

KeenKeeper is a personal relationship management web application designed to help users keep track of meaningful connections. It provides a simple way to browse friends, manage interaction timelines, and maintain important relationships through an organized and responsive interface.

## 🔗 Live Site
👉 [https://shoyon-at-netlify-keen-keeper.netlify.app](https://shoyon-at-netlify-keen-keeper.netlify.app)

## 🛠 Technologies Used
- Next.js
- React.js
- Tailwind CSS
- DaisyUI
- React Icons
- LocalStorage

## ✨ Key Features
- **Friend Directory:** Browse and view a collection of friends with their information.
- **Relationship Timeline:** Add and track timeline events for each friend.
- **Responsive Design:** Smooth and clean UI that works across devices.
